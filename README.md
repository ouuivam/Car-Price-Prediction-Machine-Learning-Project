## This Streamlit project displays a price estimation in Moroccan Dirhams (MAD) for the local market.
- **Language:** French  
- **Target Market:** Morocco  
- **Main Feature:** Shows the estimated price in a clean and user-friendly web interface using Streamlit.  
- **Purpose:** Provide users with a quick and visual price estimate tailored to the Moroccan context.
## Requirements 
- Python 3.8 or higher  
- Streamlit (`pip install streamlit`)  
- Optional: Web browser to view the Streamlit app  
- Basic familiarity with running Python scripts
