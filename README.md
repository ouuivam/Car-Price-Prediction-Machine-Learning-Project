This Streamlit project provides an estimated price for cars in Morocco, displayed in Moroccan Dirhams (MAD).
- **Language:** French  
- **Target Market:** Morocco (Used Car Market)  
- **Main Feature:** Displays the estimated price of a used car in a clean and user-friendly web interface using Streamlit.  
- **Purpose:** Help users quickly get a realistic price estimate for used cars based on local market trends.
## Requirements 
- Python 3.8 or higher  
- Streamlit (`pip install streamlit`)  
- Optional: Web browser to view the Streamlit app  
- Basic familiarity with running Python scripts
